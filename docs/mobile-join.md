# 모바일 회원가입 페이지

## 개요

모바일 기기에서 회원이 직접 가입할 수 있는 공개 페이지입니다. 관리자 시스템(사이드바)과 분리된 독립적인 레이아웃으로 제공됩니다.

## 접근 방법

### URL

```
https://[도메인]/join
```

로컬 개발 환경:

```
http://localhost:3000/join
```

### QR 코드 활용

모바일 사용자의 편의를 위해 `/join` URL을 QR 코드로 생성하여 활용할 수 있습니다.

**QR 코드 이미지:** `/public/images/join-qr.png`

![회원가입 QR 코드](/images/join-qr.png)

QR 코드 재생성 방법:

```bash
node scripts/generate-qr.js "https://your-domain.com/join"
```

## 가입 흐름

5단계 위자드 형태로 진행됩니다:

```
[1단계] → [2단계] → [3단계] → [4단계] → [5단계] → [완료]
기본정보    주소      회원정보    회비선택     동의       성공
```

### 1단계: 기본정보

| 필드         | 필수 | 설명             |
| ------------ | ---- | ---------------- |
| 성명         | O    | 회원 이름        |
| 생년월일     | O    | 나이 검증에 사용 |
| 성별         | O    | 남성/여성 선택   |
| 휴대전화     | O    | 연락처           |
| 이메일       | O    | 중복 검사 수행   |
| SMS 수신동의 | -    | 토글 선택        |

### 2단계: 주소

| 필드 | 필수 | 설명          |
| ---- | ---- | ------------- |
| 주소 | O    | 우편물 발송용 |

### 3단계: 회원정보

| 필드     | 필수 | 설명                     |
| -------- | ---- | ------------------------ |
| 회원구분 | O    | 정회원/준회원/청소년회원 |
| 출석교회 | -    | 선택 입력                |
| 직분     | -    | 선택 입력                |

**나이 기반 자동 필터링:**

- 정회원/준회원: 만 19세 이상
- 청소년회원: 만 13~18세

### 4단계: 회비선택

> 청소년회원은 이 단계를 건너뜁니다.

| 유형 | 금액                |
| ---- | ------------------- |
| 월납 | 10,000원/월         |
| 연납 | 100,000원/연 (추천) |
| 평생 | 1,000,000원         |

**입금 계좌 안내:**

- 은행: 국민은행
- 계좌번호: 483901-01-188268
- 예금주: 성서유니온선교회

### 5단계: 동의 및 확인

| 항목                                 | 필수           |
| ------------------------------------ | -------------- |
| 개인정보 수집·이용 동의              | O              |
| 마케팅 정보 수신 동의                | -              |
| 마케팅 수신 방법 (문자/이메일/둘 다) | 마케팅 동의 시 |

입력 정보 요약이 함께 표시됩니다.

## 가입 완료 후

### 완료 페이지 (`/join/success`)

- 가입 완료 메시지
- 회원 구분 배지 표시
- (청소년 제외) 입금 계좌 안내
- 이메일 확인 안내

### 자동 처리 항목

1. **회원 생성**: DB에 회원 정보 저장
2. **회비 생성**: 선택한 유형에 따라 자동 생성
   - 월납: 가입 월 ~ 12월까지 UNPAID 레코드
   - 연납: 해당 연도 1건
   - 평생: 1건
3. **감사 로그**: 가입 이력 기록

### 관리자 가입과의 차이점

| 항목       | 모바일 가입 (`/join`) | 관리자 가입 (`/members/new`) |
| ---------- | --------------------- | ---------------------------- |
| 접근 권한  | 공개                  | 관리자 전용                  |
| 레이아웃   | 단독 (사이드바 없음)  | 관리자 레이아웃              |
| 결제 방식  | 직접입금만            | 직접입금 + CMS               |
| PDF 업로드 | 불가                  | AI 자동 인식 지원            |
| 폼 형태    | 5단계 위자드          | 단일 페이지                  |

## 주의사항

1. **CMS 자동이체 미지원**: 모바일 가입은 직접입금만 지원합니다. CMS는 관리자가 회원 정보 수정을 통해 추후 등록합니다.

2. **이메일 중복 검사**: 이미 등록된 이메일은 가입 불가합니다.

3. **나이 검증**: 생년월일 기준으로 회원구분 조건이 자동 검증됩니다.

4. **입금 확인**: 회원이 직접 입금해야 하며, 관리자가 납부 처리를 수동으로 진행합니다.
