# 회원가입 신청서 PDF 업로드 자동 등록 기능

## 개요

회원 등록 페이지(`/members/new`)에서 회원가입 신청서 PDF 또는 이미지 파일을 업로드하면, Claude Vision API가 내용을 자동으로 추출하여 회원 등록 폼을 채워주는 기능입니다.

---

## 주요 기능

| 기능              | 설명                                    |
| ----------------- | --------------------------------------- |
| PDF/이미지 업로드 | 회원가입 신청서 파일 업로드             |
| AI 데이터 추출    | Claude Vision API로 문서 내용 자동 인식 |
| 폼 자동 채우기    | 추출된 데이터로 입력 필드 자동 입력     |
| 검토 후 등록      | 사용자가 내용 확인/수정 후 최종 등록    |

---

## 지원 파일 형식

| 형식 | MIME Type         | 비고             |
| ---- | ----------------- | ---------------- |
| PDF  | `application/pdf` | 다중 페이지 지원 |
| PNG  | `image/png`       | -                |
| JPEG | `image/jpeg`      | -                |
| WebP | `image/webp`      | -                |

- **최대 파일 크기**: 10MB

## 사용 방법

### 1. 파일 업로드

회원 등록 페이지 상단의 업로드 영역에서 파일 선택

```
┌─────────────────────────────────────────────┐
│  📄 회원가입 신청서 PDF 업로드 (선택사항)    │
│  ┌─────────────────────────────────────┐   │
│  │  파일을 클릭하여 선택하거나         │   │
│  │  드래그하세요                       │   │
│  │  PDF, PNG, JPG (최대 10MB)          │   │
│  └─────────────────────────────────────┘   │
└─────────────────────────────────────────────┘
```

### 2. 자동 채우기 실행

파일 선택 후 "자동 채우기" 버튼 클릭

```
┌─────────────────────────────────────────────┐
│  📄 신청서.pdf                      [X]    │
│  125.3 KB                                   │
│  ┌─────────────────────────────────────┐   │
│  │        🔄 자동 채우기               │   │
│  └─────────────────────────────────────┘   │
└─────────────────────────────────────────────┘
```

### 3. AI 처리 중

Claude Vision API가 문서 내용을 분석

```
┌─────────────────────────────────────────────┐
│  ⏳ AI가 내용을 추출하는 중...              │
└─────────────────────────────────────────────┘
```

### 4. 결과 확인 및 수정

추출된 데이터가 폼에 자동 입력됨. 사용자가 내용 확인 후 필요시 수정

### 5. 회원 등록

"회원 등록" 버튼 클릭하여 최종 등록

## API 엔드포인트

### POST `/api/members/parse-pdf`

PDF/이미지 파일을 파싱하여 회원 정보 추출

**Request**

```
Content-Type: multipart/form-data

file: [PDF or Image File]
```

**Response (성공)**

  "data": {

    "birthDate": "1990-05-15",

    "phone": "010-1234-5678",
    "email": "hong@example.com",
    "church": "새생명교회",
    "position": "집사",
    "memberType": "REGULAR",
    "feeType": "MONTHLY",

      "bankName": "국민은행",

      "accountHolder": "홍길동",

## 제한사항

1. **파일 크기**: 최대 10MB
2. **지원 형식**: PDF, PNG, JPEG, WebP만 지원
3. **인식 정확도**: 문서 품질에 따라 다를 수 있음
4. **필수 검토**: 추출된 데이터는 반드시 사용자가 확인 필요

---

## 트러블슈팅

### 파싱 실패 시

1. 파일이 손상되지 않았는지 확인
2. 파일 크기가 10MB 이하인지 확인
3. 문서가 선명하게 스캔되었는지 확인
4. 다시 시도하거나 직접 입력

### 데이터 누락 시

- AI가 인식하지 못한 필드는 수동 입력 필요
- 체크박스 항목은 직접 확인 권장
