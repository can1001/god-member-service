## 1. 암호화 함수 deprecated API 수정

### 1.1 문제 상황

**증상**: 회원 등록 시 아래 오류 발생

```
TypeError: b.default.createCipher is not a function
암호화 실패: 데이터 암호화에 실패했습니다.
```

**원인**: Node.js에서 `crypto.createCipher()` 및 `crypto.createDecipher()` 함수가 보안상의 이유로 deprecated되어 제거됨

### 1.2 수정 내용

**파일**: `src/lib/crypto.ts`

| 위치       | 변경 전                                 | 변경 후                                                                                 |
| ---------- | --------------------------------------- | --------------------------------------------------------------------------------------- |
| 38번 줄    | `crypto.createCipher(ALGORITHM, key)`   | `crypto.createCipheriv(ALGORITHM, key, iv)`                                             |
| 76-77번 줄 | `crypto.createDecipher(ALGORITHM, key)` | `const iv = Buffer.from(ivHex, 'hex')`<br>`crypto.createDecipheriv(ALGORITHM, key, iv)` |

### 1.3 변경 코드

**encrypt() 함수**:

```typescript
const cipher = crypto.createCipheriv(ALGORITHM, key, iv)
```

**decrypt() 함수**:

```typescript
const iv = Buffer.from(ivHex, 'hex')
const decipher = crypto.createDecipheriv(ALGORITHM, key, iv)
```

### 1.4 영향 범위

- CMS 자동이체 회원 등록 시 계좌번호/생년월일 암호화
- 빌링키 암호화/복호화
